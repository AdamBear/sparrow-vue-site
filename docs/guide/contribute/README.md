# 运行源代码
[源代码在 sparrow-js/packages下](https://github.com/sparrow-js/sparrow/tree/master/packages)

- sparrow-server 服务端代码
```bash
// 进入sparrow-server
$ yarn
$ yarn dev
```
- sparrow-view 预览服务
```bash
// 进入sparrow-view
$ yarn
$ yarn serve
```

- sparrow-client 操作界面

```bash
// 进入sparrow-client
$ yarn
$ yarn serve
// 访问 http://localhost:8080/
```
